# Home Compass · 让每个普通人在买房前看清真相

> 用公开数据,把北京板块、成交、政策一次说清楚。30 秒看懂一个板块到底值不值。

[**🌐 Live Demo**](https://home-compass-eight.vercel.app) · [GitHub](https://github.com/heggria/home-compass) · MIT

不替你做决定,只帮你看清差距。MVP 聚焦北京,后续再考虑扩展。

## 它做什么

- 「能买多少钱的房」计算器:输入收入/首付/利率/年限,反推总价上限
- 板块打分卡:综合分按 普通买房人 视角加权
  - 可负担 30 · 通勤 20 · 政策 20 · 流动性 15 · 学区 15
- 每个板块给出亮点 / 风险摘要,辅助快速过筛

## 它不做什么

- 不预测房价涨跌
- 不替任何中介、开发商、运营做引流
- 不展示个人房源,所有数据均为公开统计或脱敏成交

## 本地开发

```bash
pnpm install
pnpm dev          # http://localhost:3000
pnpm build && pnpm start
```

要求 Node >= 20。

## 部署

仓库已配置 Vercel 一键部署:推送到 `main` 自动构建。

## 数据来源 & 免责

- 板块均价、同比、亮点 / 风险均来自公开渠道与作者本人粗估,**仅供研究**。
- 所有数字都不构成投资建议;请以官方与中介现场核实为准。
- 欢迎通过 Issue / PR 修正数据偏差。
